Para poder ver a visualização de DadosLegislativas.py, o ficheiro DadosLegislativas.py tem que
ser corrido em um bokeh server.
Eu instalei bokeh com anaconda em um novo environment. A minha maneira de abrir
um bokeh server
1.abrir anaconda prompt
2.escrever: conda activate environment
3.no prompt ir para a folder onde o DadosLegislativas.py é guardado (cd Documents\..)
4.escrever: bokeh serve --show DadosLegislativas.py
